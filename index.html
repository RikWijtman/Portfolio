Introductie

Goedenmorgen allemaal, ik ben Rik Wijtman momenteel 16 Jaar oud,
Ik zit op de Hogenschool Rotterdam en doe de studie Creative Media en Game Technologies(CMGT)

Mijn hobbies

Ik heb vele hobbies onder andere tekenen, tennissen en games ontwerpen. Ik Teken vooral Pokemon en horror wezens, deze zijn te zien op mijn instagram en tiktok. Ik zit momenteel zes jaartjes op tennis en kan er nogsteeds vrij weinig van, Ik doe dit elke donderdag avond met een groepje vrienden. Ik heb vorig jaar mijn eerste game ontworpen, het heet Turtlecrab, het is een simpele platformer met als einde een boss battle. De game is helaas nooit uitgebracht op steam of andere websites. Mijn andere hobbies zijn duiken, bowlen en gamen.

Wat wil ik leren

Ik wil graag tijdens deze opleiding leren om nog beter te programeren en games te kunnen ontwerpen. Ik hoop tijdens deze opleiding met een paar mede studenten een game te ontwerpen die ook echt te downloaden is op steam. Het is me nu al gelukt om Pokemon te programeren op de adafruit playground, dus ik durf te wedden dat een echte game maken ook moet lukken.

Wat heb ik geleerd

Tijdens de opleiding heb ik drie vakken gehad: Programeren, Front end development en ontwerpen. Bij programeren werken we met een circuit playground waarop je met javascript kan programeren, zo heb ik tijdens dit vak veel javascript geleerd zoals arrays en mapping. Front end development was toch wel mijn lastigste vak aangezien ik niks van HTML en CSS wist, zo heb ik hier geleerd hoe je je website opdeelt in verschilende gedeeltes, hoe je flex gebruikt en hoe je css in koppeling werkt met de html. Bij ontwerpen leer je vooral over het proces van het maken van een product, zo heb ik geleerd hoe je het best het maken van een product kan aanpakken en in welke stappen.

Mijn projecten

Momenteel ben ik bezig met het maken van een kat wekker, de wekker maakt de persoon op een natuurlijke manier wakker door steeds feller en steeds harder geluid te maken.
